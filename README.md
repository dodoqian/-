# -
题目复习程序

项目结构

\exam\-\
├── deepseek_json_20260519_1e3487.json   # 题库源文件（300 判断 + 300 选择）
├── build_quiz.py                         # HTML 生成脚本
└── quiz.html                             # 答题工具（自包含单文件）
quiz.html 功能概览
功能	说明
三种模式	判断题 / 选择题 / 混合模式，可自由切换
随机抽题	从题库中随机抽取，每局不重复
题目数量	可选 10 / 20 / 30 / 50 / 100 题 / 全部
即时反馈	选择后立即显示对错、正确答案和解析
进度条	实时显示答题进度和得分
结果页	百分比评分、正确/错误统计、逐题回顾
错题本	自动收录答错的题，支持按类型筛选和逐条移除，数据持久化在 localStorage
使用方式
直接用浏览器打开 quiz.html 即可，无需服务器或任何依赖。重新生成可运行 python build_quiz.py。
